포트폴리오

소개

프리랜서는 Start Bootstrap에서 만든 무료 부트스트랩 테마입니다. 다운로드에는 HTML, CSS, JavaScript를 포함한 전체 소스 파일이 포함되어 있으며, 커스터마이징이 용이한 선택적 SASS 스타일시트도 포함되어 있습니다.

머리글에 사용할 나만의 맞춤형 아바타를 만들고, 구분선 아이콘을 변경하며, 이메일 주소를 추가하여 연락처 양식을 완전히 기능적으로 만들 수 있습니다!
